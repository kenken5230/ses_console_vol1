## 6. 誤分類しやすい表現

### 6.1 `要員` という単語があっても案件寄りのもの

以下は `要員` を含むが、「要員を探している」「要員を募集している」ため、案件寄りとして扱う。

- `要員探してます`
- `作業要員`
- `開発作業要員`
- `I-Filter変更作業要員`
- `2名募集`
- `メンバー募集`
- `要員募集`

例:

- `[SES配信] 【急募！確度高いです！】福岡のJava要員探してます！Java／PG／SE／PL／面談1回`
- `[SES配信] 【確度高いです！】VMware要員探してます！VMware／マネジメント／ベンダーコントロール`
- `[SES配信] 【ARI】I-Filter変更作業要員/7月～`

### 6.2 `フリーランス` / `個人事業主` は本文確認が必要

`弊社直フリーランス` や `弊社直個人事業主` は要員紹介である可能性が高いが、案件名の一部として使われる可能性もある。

初期ルールでは要員寄りに寄せる。ただし、本文内に `募集`、`案件内容`、`作業内容`、`面談`、`精算`、`商流` が強く出る場合は案件へ倒す。

## 8. 次に追加するルール候補

### 8.1 要員ルール候補

- 件名または本文に `人材情報` / `要員情報` / `正社員人材` がある場合は要員紹介を強く加点。
- 件名または本文に `個人事業主` / `フリーランス` があり、かつ `ご紹介` / `稼働実績` / `経歴` / `希望単価` / `スキルシート` / `最寄` がある場合は要員紹介。
- `CTO/PM要員` のように人物属性としての要員が使われている場合は要員紹介。

### 8.2 案件ルール候補

- 件名に `募集` / `支援` / `設計` / `構築` / `開発` / `導入` / `移行` / `運用` があり、開始月・勤務地・面談・単価・リモート等の条件語がある場合は案件紹介。
- `要員探してます` / `作業要員` / `メンバー募集` / `2名募集` は案件紹介。
- `現場直` / `元請直` / `増員枠` / `交代枠` は案件紹介を加点。

### 8.3 除外ルール候補

- `ご招待` + `交流会` は除外。
- `セミナー` / `ウェビナー` / `イベント` / `勉強会` は除外。
